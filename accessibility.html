Our Commitment

Kedar Oak Research Analyst (SEBI Reg. No.: INH000001055), operated by Kesar Securities, is committed to ensuring digital accessibility for people with disabilities. We continually improve the user experience for everyone and apply the relevant accessibility standards to ensure we provide equal access to all users.

Conformance Standards

This website aims to conform to:

  • WCAG 2.1 Level AA — Web Content Accessibility Guidelines by W3C
  • GIGW 3.0 — Guidelines for Indian Government Websites
  • IS 17802 — Indian Standard for ICT Accessibility

Technologies Used

This website is built using the following technologies:

  • HTML5
  • CSS3
  • JavaScript (ES6+)
  • Bootstrap 5
  • AOS.js (Animate on Scroll)
  • Font Awesome (icons)

Accessibility Features

The following accessibility features have been implemented across this website:

  • Skip Navigation — A "Skip to main content" link is provided on every page, allowing keyboard and screen reader users to bypass repetitive navigation.
  • Keyboard Navigation — All interactive elements (links, buttons, form fields, menus) are fully accessible via keyboard. The hamburger menu and compliance dropdown support Enter, Space, and Escape keys.
  • Screen Reader Compatibility — Semantic HTML landmarks (<header>, <nav>, <main>, <footer>) are used throughout. ARIA labels, roles, and states are applied to interactive components.
  • Focus Indicators — Clearly visible focus outlines are provided on all interactive elements for keyboard users.
  • Text Resize Support — An accessibility toolbar provides text size controls (A, A+, A++) that persist across pages. The site also supports browser zoom up to 200% without loss of content.
  • High Contrast Mode — A high contrast toggle is available in the accessibility toolbar for users who need enhanced visual contrast.
  • Alternative Text — All meaningful images include descriptive alternative text. Decorative images are marked with empty alt attributes and aria-hidden.
  • Form Accessibility — The contact form includes proper labels, required field indicators, autocomplete attributes, and accessible error messages with ARIA live regions.
  • Reduced Motion — Animations are automatically disabled for users who have enabled the "Reduce Motion" preference in their operating system.
  • Read Aloud (Text-to-Speech) — A "Read Aloud" button in the accessibility toolbar uses your browser's speech synthesis to read page content aloud, highlighting each section as it is read. Helpful for blind and visually impaired users who may not have a dedicated screen reader.
  • Dyslexia-Friendly Font — A toggle in the toolbar switches all text to a dyslexia-friendly font with increased letter and word spacing for improved readability.
  • Grayscale Mode — Removes colour from the page, which can help users with certain visual or cognitive conditions.
  • Reading Guide — A horizontal guide bar follows the mouse cursor to help users track their reading position across lines of text.
  • Line Spacing Control — Increases line height and paragraph spacing for easier reading.
  • Consistent Navigation — Navigation menus appear in the same order and location on every page.
  • Descriptive Page Titles — Each page has a unique, descriptive title tag.
  • Proper Heading Structure — Pages use a logical heading hierarchy (h1 through h6) without skipping levels.

Known Limitations

Despite our best efforts, some areas may not yet be fully accessible:

  • PDF Documents — Compliance documents (Investor Charter, Investor Complaints, MITC, Terms and Conditions) are provided as PDF files. These may not be fully accessible to screen readers. We are working to improve their accessibility. If you need the content of any PDF in an accessible format, please contact us.
  • Third-Party Content — Some embedded content or third-party services may not fully meet accessibility standards.
  • Gallery Images — Gallery photos have generic descriptive alt text. We are working to provide more detailed descriptions.
